# DemocracyConnect

Indian Election System Education for Users

## Overview
This interactive application is designed to educate citizens about the Indian election process. It features a "Tiranga"-themed UI, an AI chatbot, interactive flashcards, quizzes, and a visual timeline of the election process.

## Technologies
- React + Vite
- Framer Motion for animations
- i18next for multi-language support (22 Indian languages)
- Docker for containerization
